//Blinkit  App (E -Commerce/Grocery)
--------------------------------------------------------------------------------------------------------------------------------------------------
●	Description: This project is the complete User Interface for Blinkit (India's Last Minute App), a quick commerce platform promising 16-minute delivery.
The UI covers the entire user flow: Onboarding/Login; product discovery on the Home Screen; a seamless Cart Screen; filtered Category Views;
and an expanded service via the Print Store feature for document uploads. 
The design is fully prepared for real-time data fetching and order processing.
--------------------------------------------------------------------------------------------------------------------------------------------------
●	Technologies: Dart, Flutter (for cross-platform UI), Third-Party REST API (Planned for backend integration)

●	IDE Used: Android Studio

●	Duration: Developing Phase
